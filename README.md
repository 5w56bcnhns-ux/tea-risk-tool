專案說明書：「茶言觀色」顧客行為分析與流失預警
1. 專案背景 (Project Background)
隨著手搖飲市場競爭加劇，「茶言觀色」品牌希望透過數據驅動決策，深入了解顧客的消費習性。本專案旨在透過分析現有的顧客數據，找出影響顧客流失（Churn）的關鍵因素，並針對不同行政區與產品偏好制定精準的行銷策略。

2. 專案目標 (Project Objectives)
數據優化： 清理原始數據中的混亂格式與缺失值，建立標準化分析庫。

流失診斷： 量化顧客流失率，並找出流失與消費行為之間的相關性。

客群畫像： 分析不同性別、地區及產品偏好顧客的貢獻度。

行動建議： 根據數據結論提出提升留存率（Retention）的具體方案。

3. 數據規格與處理 (Data Scope & Methodology)
3.1 數據概況

樣本數： 302 筆有效顧客記錄。

維度： 涵蓋顧客人口統計（年齡、性別、地區）、消費行為（月均訪次、半年總額、產品偏好）以及數位參與度（App 使用、促銷參與）。

3.2 關鍵處理流程

資料標準化： 將 10 多種性別標籤（如：f, Female, 女）統一歸併。

異常值填補： 針對 age、satisfaction 等欄位使用中位數填補，確保統計不失真。

關聯性運算： 使用皮爾森相關係數（Pearson Correlation）探討流失（Churned）與其他變量間的關係。

4. 核心分析發現 (Key Findings)
A. 流失風險預警

整體流失率： 25.2%，意即每四位顧客就有一位可能不再回購。

關鍵影響因子：

數位工具： 使用 App 的顧客流失率顯著較低（相關係數 -0.21）。

活躍度： 距離上次消費天數（Recency）是流失的最強預測指標。

B. 產品偏好表現

忠誠度冠軍： 咖啡系列。其受眾流失率僅 11.9%，代表該品類具備極強的顧客黏性。

品質預警區： 水果茶與特調飲品。流失率高達 33.3%，顯示這兩類產品在口味穩定度或新鮮感維持上存在挑戰。

C. 地域貢獻度

大安區 為核心戰區，半年貢獻額破 20 萬，遠領先信義區與士林區。

5. 策略建議 (Strategic Recommendations)
第一階段：數位轉型與留存

強制/獎勵下載 App： 數據證明 App 能有效降低流失。建議在店面推行「下載 App 即享當杯 85 折」活動。

自動化喚醒： 針對「超過 30 天未到店」的顧客，透過 App 推送專屬回流優惠券。

第二階段：產品優化

水果茶研發改善： 針對流失率高的水果茶客群發放電子問卷，確認是甜度、新鮮度還是價格問題導致流失。

第三階段：區域經營

大安區 VIP 計劃： 針對高貢獻區建立等級會員制，強化大安區顧客的尊榮感，穩固核心營收。

語言： Python 3

數據處理： Pandas, NumPy

分析維度： 描述性統計、相關性分析、交叉分析
